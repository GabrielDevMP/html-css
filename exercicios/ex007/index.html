Principais formatações

Negrito / destaque

nessa frase temos um termo em negrito usando a < b > ( sem semantica).

Nesta frase temos termo em destaque usando a tag <strong >(com semantica)

Itálico / Ênfase

Nesta frase temos um termo em Italico usando a tag <I> (sem semantica)

Nesta frase temos um termo em destaque usando a tag <EM> (com semantica)

Texto Marcado

podemos criar também um texto marcado usando a tag MARK

e no outro parágrafo temos um outro termo marcado no final

Texto grande e pequeno

estamos criando um texto grande com a tag <big > (sem semantica) e um texto pequeno com a tag <small>a nesse parágrafo

Texto deletado

podmos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag <del>

Texto Inserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. com a tag < ins >

Existe também o sublinhado com a tag <U > (sem semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag < SUP >

Texto Subscrito

para inserir coisas do tipo H2O com a tag < SUB >
